Elements of a marathon training plan

Cross-training

Cross-training can include a range of physical activities, such as using elliptical machines, biking or swimming. Make sure it’s an activity that you enjoy.

Pace Runs

Start these runs with a warmup by walking or slowly jogging. Then transition to running a set pace. With each pace run you do, try each week to get closer to your goal pace for your marathon.

Rest Days

After you put in all these miles, you’re going to need to take a break.
